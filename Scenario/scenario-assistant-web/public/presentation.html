설정의 일관성을 잃기 쉬운 창작자들을 위해

소설 및 시나리오 작성 프로세스에서, 등장인물의 정보와 플롯의 전개, 배경 장소가 늘어남에 따라 설정의 일관성을 지키는 것은 대단히 어렵습니다.

본 플랫폼은 창작자가 텍스트 설정을 일일이 기억하고 관리해야 하는 복잡성을 해결하기 위해 제작되었습니다. 로컬 기기 내에서 동작하는 경량화 파인튜닝(SFT) 모델을 탑재하여, 작품 텍스트만 주입하면 자동으로 사건과 세계관을 관계망 형태로 추출하고, 일관성을 유지하도록 돕는 창작 조력자를 지향합니다.

01. AI 스토리 데이터 추출

시나리오 원문을 붙여넣으면 LLM(설정 모델)이 등장인물, 장소, 핵심 플롯을 규격화된 포맷으로 고정 정밀 추출하고 자동으로 인과관계를 태깅합니다.

02. 대화형 세계관 관리 보드

추출된 데이터를 바탕으로, 보드 인터페이스 상에서 캐릭터의 세부 특징을 변경하고 관계 맵을 수정 및 추가할 수 있는 에디터를 제공합니다.

03. 세계관 설정 무결성 검사

전체 세계관을 LLM이 상호 크로스 텍스트 분석하여, 모순된 설정이나 시간선 오류, 캐릭터 중복을 선제적으로 찾아 경고합니다.

04. GraphRAG 세계관 질의

Microsoft GraphRAG에 기반하여 로컬 벡터 DB(ChromaDB)에 수립된 하이브리드 지식 그래프를 탐색, 세부 질문에 대해 고정밀 정보를 제공합니다.

로컬 SFT 파이프라인 구축

  • 대상 모델 검토: Qwen, Gemma, EXAONE, DeepSeek 등 경량 온디바이스 모델 탐색.
  • 정답지 구축: Claude 3.5 Haiku를 활용하여 200개 소설 청크의 구조화 정답 데이터셋 구축.
  • 학습 환경 최적화: LoRA 파라미터 튜닝, VRAM 효율성 극대화를 위한 Target Module 매핑 및 모델 캐시 드라이브 경로 최적화.

학습 대상 모델별 특징 및 평가

  • Qwen: 구조화 능력이 매우 우수하여 SFT 진행 후 JSON 스키마 준수율 100% 및 최고 F1-Score(0.5294) 기록.
  • Gemma: 한국어 문맥 이해가 뛰어나나 JSON 키 혼용 가능성 식별 문제가 있었으나,필터 적용으로 에러율 2.5% 미만으로 감소
  • DeepSeek: 스키마 일관성은 우수하나 추론 필터링 및 결측치 방지를 위한 추가적인 프롬프트 엔지니어링 필요성 확인.
  • EXAONE: 문맥 묘사는 자연스러우나 온디바이스 환경에서 파라미터 크기 대비 JSON 출력 안정성의 편차 존재.
