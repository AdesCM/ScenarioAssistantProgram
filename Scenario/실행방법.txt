# Scenario Assistant Application 실행 방법

이 프로젝트는 웹 프론트엔드(React/Vite)와 GraphRAG 파이썬 백엔드(FastAPI) 두 부분으로 구성되어 있으며, 로컬 AI 실행을 위한 Ollama가 필요합니다.

---

## 1. 전제 조건 (필수 설치 항목)
- **Node.js** (v18 이상 권장)
- **Python** (v3.10 이상 권장)
- **Ollama**: 로컬에서 LLM 모델(예: `gemma2`)을 처리하기 위해 필요합니다.

---

## 2. 파이썬 백엔드 (RAG 서버) 실행 방법

GraphRAG와 벡터 DB(ChromaDB)를 구동하는 백엔드 서버입니다.

1. 터미널을 열고 `rag-server` 폴더로 이동합니다.
   ```bash
   cd e:\Work\ScenarioAssistantApplication\ScenarioAssistantApplication\Scenario\rag-server
   ```
2. (선택) 파이썬 가상환경을 생성하고 활성화합니다.
   ```bash
   python -m venv venv
   # 윈도우 환경:
   venv\Scripts\activate
   # Mac/Linux 환경:
   source venv/bin/activate
   ```
3. 필요한 패키지를 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
4. 서버를 실행합니다.
   ```bash
   python main.py
   # 또는 uvicorn main:app --reload
   ```
5. 서버가 정상적으로 실행되면 `http://127.0.0.1:8000` 에서 구동됩니다.

---

## 3. 프론트엔드 (웹 앱) 실행 방법

Vite 환경에서 돌아가는 React 애플리케이션입니다.

1. 새로운 터미널을 열고 `scenario-assistant-web` 폴더로 이동합니다.
   ```bash
   cd e:\Work\ScenarioAssistantApplication\ScenarioAssistantApplication\Scenario\scenario-assistant-web
   ```
2. 필요한 패키지를 설치합니다.
   ```bash
   npm install
   ```
3. 프론트엔드 서버를 실행합니다.
   ```bash
   npm run dev
   ```
4. 터미널에 표시되는 로컬 서버 주소(보통 `http://localhost:5173/`)를 웹 브라우저에 입력하여 접속합니다.

---

## 4. 로컬 AI 서버 (Ollama) 실행 (필요 시)

애플리케이션 내의 "AI 도구" 기능을 정상적으로 사용하려면 Ollama가 백그라운드에서 동작하고 있어야 합니다.
웹 UI 설정이나 코드에 지정된 모델(예: `gemma2:latest`)이 Ollama에 다운로드되어 있어야 합니다.

- 다운로드 및 실행 예시 (cmd 또는 PowerShell에서)
  ```bash
  ollama run gemma2
  ```

---

### **종합 실행 요약**
- 터미널 1: `ollama`가 실행 중인지 확인
- 터미널 2: `rag-server` 폴더에서 `python main.py` 실행
- 터미널 3: `scenario-assistant-web` 폴더에서 `npm run dev` 실행
